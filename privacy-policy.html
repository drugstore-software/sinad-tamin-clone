Privacy Policy for Sinad Tamin Clone

Effective Date: October 2025

Overview

Sinad Tamin Clone is a Chrome extension designed to help users read and copy drug information from Social Security (Tamin) electronic prescriptions.

Data Collection

This extension does not collect, store, transmit, or share any personal, medical, or browsing data. All operations happen locally within your browser.

Permissions Explanation

  • activeTab: Used to read and extract information from the currently active Tamin prescription tab when the user activates the extension.
  • clipboardWrite: Used to copy extracted prescription data to the clipboard upon user request.
  • scripting: Used to inject local scripts that read visible data from the prescription page.
  • storage: Used to save user settings locally (e.g., display preferences).
  • host_permissions: Limited to the Tamin website (https://darman.tamin.ir/*).

Remote Code

No remote code or external scripts are loaded or executed by this extension.

Third-Party Services

This extension does not use or connect to any third-party services.

Data Security

Because no data leaves the user’s browser, there are no external security risks. Users maintain full control of their information.
